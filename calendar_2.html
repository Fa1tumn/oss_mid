중강고사 대체 과제


중간고사 대체 과제는 
과제 가이드라인 https://hongbuzz.notion.site/2025_OSS_-1d1af259fd0580a3b7c6ddedcef71c52?pvs=4
에 맞춰서 

개인 및 팀별로 지정한 https://ys.learnus.org/mod/ubboard/view.php?id=4003541 (세부 주제 변경은 필요에 따라서 개별 개선 및 수정가능하다 )
과제 주제를 해결한다. 

제출형태 

1. 파일 첨부 : 보고서 (양식은 가이드라인 참조) mid_term_nlp_학번_이름_팀번호.pdf 
2. 직접 작성 : github repo url ( 공개 리포 추천, 만약 비공개의 경우 hongsw 아이디를 Collabolator로 추가 필수 )

8주차는 중간고사 제출을 대체하고,(수업없음)
9주차에 발표를 제출한 보고서를 바탕으로 진행합니다. 

평가방법 

항목배점평가 포인트
1. 아이디어 기획의 실현 가능성과 실용성25점작업 문제를 잘 정의했는가? CLI/API/전문지식 기반으로 자동화 필요성이 있는가? 기능 범위는 적절한가?
2. 구현 및 협업 과정의 충실도25점역할 분담이 명확했고, 각 역할을 성실히 수행했는가? MCP 서버의 도구와 리소스가 정상 동작하며 커밋 기록이 정돈되었는가?
3. 기술 적용 및 문서화 완성도20점FastMCP/Node.js SDK/Docker 등 기술 스택 사용이 명확하며, schema/README.md 에 사용법 등 문서가 OpenSource 프로젝트의 요구사항(소개, 설치방법, 사용방법, 라이선스 명시)을 충실한가?
4. 실행 가능성과 재현성15점다른 사용자가 그대로 따라 실행 가능한가? 도구 실행 예시, 시연 흐름이 명확히 정리되었는가?
5. 발표 내용과 협업 설명력15점기능 설명이 명확하며 팀워크와 역할 수행 과정이 잘 전달되었는가? 질문에 실용적, 기술적으로 응답했는가?
